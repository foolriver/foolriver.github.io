kubeadm搭建k8s_v1.11.1

发表于 2018-10-17

第一部分：机器前期配置

三台centos7机器：

master    192.168.10.140
node1    192.168.10.139
node2    192.168.10.141

设置主机名与IP地址的映射 （三台机器）

$ vim /etc/hosts
(如果没有vim的可以使用vi 或者yum install vim,下面是在master节点上的设置)
127.0.0.1    master
192.168.10.139    node1
192.168.10.141    node2
重启网络服务
$ systemctl restart network
